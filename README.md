# e-badge

## Setup
### Automatic

    git clone https://github.com/razurac/e-badge.git
    ./setup.sh

### Manual

    git clone https://github.com/razurac/e-badge.git
    pip3 install -r requirements.sh
    apt install libopenjp2-7 libtiff5


## Usage
### Text UI
    python3 main.py --tui
This will bring up a text UI that lets you choose various options.
